Curso Mobile Developer

Duração: 360 horas

Professores: Maria Dolores, Antonio Manoel, Guilherme Caucaia

Descrição: O curso Mobile Developer tem como objetivo proporcionar a todos se tornar um desenvolvedor de sistemas web. O curso abordará sobre o desenvolvimento Back-End de sistemas web.

Conteúdo:

  • FUNDAMENTOS DE TECNOLOGIA DA INFORMAÇÃO
    1. Introdução aos computadores e à informática
    2. Representação de dados e sistemas de numeração
    3. O computador e suas partes
    4. Conhecer e implementar técnicas de manutenção preventiva e corretiva
    5. Processadores de texto
    6. Planilhas eletrônicas
    7. Elaboração de apresentações
  • LÓGICA DE PROGRAMAÇÃO E ORIENTAÇÃO A OBJETOS JAVA
    1. Algoritmos: Conceito, Tipos de dados
    2. Variáveis
    3. Operadores
    4. Estruturas
    5. Orientação a Objetos: Paradigma da orientação a objetos, Classes, Objetos, Atributos, Métodos, Encapsulamento, Herança, Polimorfismo, Modificadores e Interfaces
  • PROGRAMAÇÃO DE DISPOSITIVOS MOVEIS
    1. Introdução ao Android
    2. Arquitetura Android
    3. Características da Plataforma
    4. Emuladores
    5. Ferramentas
    6. Histórico
    7. Instalação e Configuração do Ambiente: Layout de tela, Ações para controle de tela, . Lista de valores para aplicação, Diálogos com data e hora, Notificações e alarmes, Mapas e GPS, Multimídia
  • BANCO DE DADOS PARA DISPOSITIVOS MOVEIS
    1. Operações em banco de dados
    2. Acesso a dados através de conexões WEB
    3. Armazenamento de dados
